Настройка динамической маршрутизации (OSPF) на RTR в HQ и BR.

Сведения занести в отчёт:

1. Для настройки был выбран OSPF (link state) протокол

2. Конфигурацию FRR (/etc/frr/frr.conf или вывод show running-config из vtysh)

3. Доказательство установки соединения (show ip ospf neighbor из vtysh, где видно состояние Full)

4. Подтверждение получения маршрутов \*\*(\*\*show ip route ospf, где видны сети противоположного офиса)

5. Подтверждение защиты протокола \*\*(\*\*show ip ospf interface tun1 из vtysh, в котором должна присутствовать строка Cryptographic authentication или Simple password authentication и указание, что ключ установлен)

6. Демонстрация связности (вывод ping)
